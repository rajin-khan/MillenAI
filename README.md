**MillenAI** is a sleek, high-performance, open-source chat interface designed for seamless interaction with multiple Large Language Models. Built with modern web technologies, it prioritizes speed, user experience, and intelligent conversation management.

<img src="./Documentation/ss1.png" alt="MillenAI Interface Screenshot" width="90%">

---

## The Problem

The landscape of Large Language Models is rapidly expanding, but interacting with them remains fragmented and frustrating:

- **Fragmented Experiences** ‚Äî Each model provider has its own interface, forcing users to juggle multiple tabs and learn different UIs
- **Lost Conversations** ‚Äî Most playgrounds are stateless, meaning valuable chat history and insights vanish when tabs close  
- **Hidden Advanced Features** ‚Äî Powerful capabilities like web search and deep reasoning require complex API integration
- **Poor User Experience** ‚Äî Functional but unpolished interfaces that lack speed and intuitive design

## Features

---

### Core Capabilities

- **Multi-Model Support** ‚Äî Seamless switching between Llama 3.1, Llama 3.3, and GPT-OSS models
- **Universal Web Search** ‚Äî Enable real-time internet access for any model with a simple toggle
- **Deep Reasoning Mode** ‚Äî Advanced problem-solving capabilities for complex, multi-step analysis
- **Secure Authentication** ‚Äî Google Sign-In with Firebase for safe conversation management
- **Real-Time Chat History** ‚Äî Save, search, rename, and organize all your conversations
- **Collapsible Reasoning** ‚Äî Clean final answers with expandable detailed thought processes
- **Live Token Counter** ‚Äî Visual progress bar for context window usage monitoring
- **Rich Markdown Support** ‚Äî Beautiful formatting for tables, lists, and syntax-highlighted code
- **Secure Settings** ‚Äî Local API key storage with comprehensive preference management
- **Guided Onboarding** ‚Äî Friendly setup process for new users
- **Modern Animations** ‚Äî Fluid interface powered by Tailwind CSS and Framer Motion
